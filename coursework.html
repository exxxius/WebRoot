PROG 109 Introduction to Web Development • 5 Cr.

Description:

Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects. Prerequisite: Placement by assessment into ENGL& 101, or completion of ENGL 092 or 093 with a C or better.

Details:

Class uses HTML, CSS and JavaScript.

Instructor:

Marciel Medina Mora. She is a kind, patient and understanding, and very helpful teacher. I have learned whatever she has taught us as far as I was present in the class!

Outcomes:

After completing this class, students should be able to:

  • Develop Web Pages using html elements to code a template for a web page.
  • Use variables with appropriate data-types and scope.
  • Use appropriate arithmetic, logical and comparison operators to develop complex expressions.
  • Apply inline, embedded, and external cascading style sheets (CSS) utilizing relative and absolute positioning.
  • Optimize images for web page display
  • Instantiate and use objects (including web forms and controls) and create JavaScript functions.
  • Code event-driven procedures (event-handlers).
